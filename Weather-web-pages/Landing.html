Summary: Latitude vs. X

Data gathered using the OpenWeatherMap API, was compiled and sorted from over 500 cities to determine the variations in weather patterns the closer one gets to the equator.

The dataset analyzed several factors to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and Visualizations created as part of the analysis, as well as explanations and descriptions of trends and correlations witnessed.
